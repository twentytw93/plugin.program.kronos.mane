# Kronos Maintenance

Kronos Maintenance is a program add-on for **Kodi 21.x (Omega)** designed for **LibreELEC** and **Windows** systems. 
It provides essential cleanup and optimization tools to keep Kodi installations running smoothly.

---

## Features
- **Thumbnail Cleanup** – Removes cached images to free up storage. 
- **Cache Management** – Clears temporary files to improve performance. 
- **Package Removal** – Deletes old add-on installation packages. 
- **Database Handling** – Resets or cleans texture databases to resolve interface glitches. 
- **Safe Execution** – All operations include error handling and abort safety. 
- **Lightweight Design** – Minimal interface with background-safe operations.

---

## Compatibility
- Kodi **21 Omega** 
- LibreELEC 12.0.2+ 
- Windows 10/11 
- Python 3 (bundled with Kodi)
